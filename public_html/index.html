Common Lisp package management in Common Lisp.

With Lispy, a user can obtain a list of installable packages, install/upgrade packages automatically and have an ASDF configuration written for them. It is a cross-platform, cross-implementation, distributed package manager written entirely in Common Lisp.

The current Lispy Map. There is also an Experimental Lispy Map which contains a map of unstable or not tested with Lispy packages.

Download

Bootstrapped Lispy releases which contain all the necessary libraries to run Lispy can be found in the releases directory.

The current bootstrapped release is 0.3.

Installation

Download and unpack the latest Bootstrapped Lispy release.

mkennedy@azathoth:~$ wget -q http://common-lisp.net/project/lispy/releases/lispy-all-0.1.tar.gz
mkennedy@azathoth:~$ tar xfz lispy-all-0.1.tar.gz 
mkennedy@azathoth:~$ ls -ld lispy-all-0.1
drwxr-xr-x 20 mkennedy mkennedy 4096 2007-07-09 22:53 lispy-all-0.1

Configure your Lisp implementation to use Lispy's ASDF configuration.

mkennedy@azathoth:~$ cat .sbclrc
(require :asdf)
(load (merge-pathnames #p"lispy-all-0.1/asdf-config.lisp" (user-homedir-pathname)))
mkennedy@azathoth:~$ sbcl

Configure Lispy variables to reflect local installation. This step is optional.

* lispy:*lispy-pathname*

#P"/home/mkennedy/lispy-all-0.1/"
* lispy:*lispy-map-url*

#<PURI:URI http://common-lisp.net/project/lispy/repository/map.lisp-expr>
*
